# 伸びやすいテーマの調査

## テーマ優先度マトリクス

テーマの選定は以下の軸で評価する：

| 評価軸 | 説明 |
|---|---|
| 🔍 検索需要 | YouTube/Google での検索ボリューム |
| 🔁 リピート性 | 毎晩・定期的にリピート再生されるか |
| 👥 ターゲット広さ | 潜在的な視聴者層の広さ |
| ⭐ 独自性 | 他チャンネルとの差別化度 |
| 💰 収益化可能性 | 有料版やパッケージ化の可能性 |

## 投稿順序（推奨）

### 最初の4本（Phase 2開始時）

1. **「考えすぎを手放す催眠誘導」** — Michael Sealey最大ヒットの日本語版ポジション
2. **「深い眠りのための催眠誘導」** — 安定需要、リピート率が高い
3. **「自己肯定感を高める催眠」** — メンタルケア系の入口
4. **「前世退行催眠体験」** — 話題性・独自性で新規流入を狙う

### その後の展開

- S/Aランクのテーマから消化していく
- 視聴者の反応（コメント・再生維持率）を見て注力テーマを絞る
- リクエストの多いテーマを追加
