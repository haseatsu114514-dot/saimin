## サムネイルのルール

- テキストは最大3行まで
- フォントサイズは画面の1/5以上
- 背景とテキストのコントラストを強く
- ロゴは右下に小さく配置
- 人物・顔は入れない（催眠の世界観を壊さない）
- テキストの背景に半透明のグラデーションを敷く

## ファイル命名規則

### 台本

```
scripts/drafts/[番号]_[テーマ英語]_[尺]min.md
scripts/published/[番号]_[テーマ英語]_[尺]min.md

例:
  scripts/drafts/001_overthinking_30min.md
  scripts/drafts/002_deep_sleep_60min.md
  scripts/published/001_overthinking_30min.md
```

### 音声ファイル

```
audio/[番号]_[テーマ英語]_[種類].wav

種類: raw（生録音）/ clean（ノイズ除去後）/ final（最終ミックス）

例:
  audio/001_overthinking_raw.wav
  audio/001_overthinking_clean.wav
  audio/001_overthinking_final.wav
```

### 動画ファイル

```
video/SHIZM_[番号]_[テーマ英語].mp4

例:
  video/SHIZM_001_overthinking.mp4
```

### サムネイル

```
thumbnails/SHIZM_[番号]_[テーマ英語]_thumb[バリエーション].png

例:
  thumbnails/SHIZM_001_overthinking_thumb_a.png
  thumbnails/SHIZM_001_overthinking_thumb_b.png
```

### 環境音・BGM素材

```
assets/bgm/[説明英語].wav
assets/sfx/[説明英語].wav

例:
  assets/bgm/ambient_meditation.wav
  assets/sfx/river_stream_loop.wav
  assets/sfx/rain_on_tent.wav
```

### 番号の付け方

- **通し番号3桁**: 001, 002, 003...（シリーズ関係なく順番）
- シリーズはフォルダではなくスプレッドシートで管理

## 動画フォーマットの統一

### シリーズ別の標準尺

| シリーズ | 標準尺 | 備考 |
|---|---|---|
| 安眠・睡眠導入 | 60分 | 長尺推奨（リピート再生でアルゴリズムに有利） |
| メンタルケア | 30分 | Michael Sealeyの主力と同じ尺 |
| 意識探求 | 45分 | 体験の深さを確保 |

### 環境音の方針

```
  ・全編に環境音を敷く（導入〜暗示は通常音量、覚醒で徐々にフェードアウト）
  ・シリーズ別の推奨環境音:
    - 安眠: 波の音 / 雨の音
    - メンタルケア: 川のせせらぎ / 森の環境音
    - 意識探求: 低音ドローン / ソルフェジオ周波数
  ・環境音の音量: 声の-15dB〜-20dB（声を邪魔しない程度）
```

### 動画の画面構成

```
  ・基本: 静止画1枚 + 音声（催眠は目を閉じて聴くため動画は不要）
  ・微妙なズーム（Ken Burns効果）を入れると滞在時間が伸びる
  ・フェーズの切り替わりで画像の明るさを微調整（深化で暗く、覚醒で明るく）
```

### エンドスクリーン

```
  ・最後の20秒をエンドスクリーン用に確保
  ・覚醒後のナレーションの後に配置
  ・次のおすすめ動画 + チャンネル登録ボタン
  ・※睡眠導入の場合はエンドスクリーンなし（視聴者は寝ている）
```
