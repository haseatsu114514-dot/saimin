### Claude vs ChatGPT の使い分け

```
  Claude                            ChatGPT
  ─────────────────────             ─────────────────────
  台本を書く           ◀──────▶     アイデアを広げる
  構成を設計する       ◀──────▶     SEO・メタデータを作る
  安全性を確認する     ◀──────▶     SNS投稿文を作る
  PC操作で実行する     ◀──────▶     マーケティング分析
                                    壁打ち・Web検索

  【原則】
  ・「台本を書く」→ Claude（リポジトリ参照）
  ・「それ以外のテキスト」→ ChatGPT
  ・「データを見て判断する」→ ChatGPT（マーケター）
  ・「実行する」→ Claude（PC画面操作）
  ・迷ったら両方に投げて良いほうを採用
```

```
       ・最終OK                   → 【自分】

【公開〜告知】（※Claude がPC操作で自動実行）

       ・公開/予約公開を設定

【振り返り】（※次の動画の工程1と兼ねてもOK）

       ・YouTube Studioのデータ取得  → Claude がPC操作でスプレッドシートに転記

       ・分析結果をスプレッドシートに記録

```

```
  ナナバナ ──(生成画像)──▶ Canva ──(テキスト配置)──▶ サムネ完成

 【音声〜動画】
  自分が収録 ──▶ Adobe Podcast ──▶ GarageBand ──▶ iMovie/DaVinci ──▶ 動画完成

 【公開〜告知】（Claude がPC画面操作で一括実行）
  スプレッドシート（メタデータ + SNS投稿文）
       │
       ▼ Claude がPC操作で読み取り
       ├──▶ YouTube Studio → アップロード・メタデータ設定・サムネ・公開

 【振り返り】（Claude がデータ収集、ChatGPT が分析）
  Claude ──(PC操作)──▶ YouTube Studio のデータをスプレッドシートに転記
       │
       ▼
  ChatGPT ──(マーケター)──▶ 傾向分析・競合調査・次回テーマ提案
       └──▶ スプレッドシートに分析結果を記録
```

## 自動化の方針

### Claude PC画面操作でできること

```
  ◎ PC操作で自動化するもの:
    ・YouTube Studio での動画アップロード・メタデータ入力・サムネ設定・公開
    ・X でのツイート投稿（スプレッドシートからコピペ）
    ・スプレッドシートの状態更新・データ記入
    ・ファイルのリネーム・整理
    ・台本のリポジトリ保存（git操作）

  ◎ PC操作で自動化するもの（マーケティング関連）:
    ・YouTube Studio の分析データ取得 → スプレッドシートへ転記
      （視聴回数・高評価・CTR・視聴維持率・コメント）
    ・YouTube Studio でのコメント確認

  ◎ PC操作で自動化を検討できるもの:
    ・Canva でのテキスト配置（テンプレが固まれば可能）

  【PC操作自動化のメリット】
    ・APIキーの取得が不要（ブラウザでログインすればOK）
    ・どのWebサービスでも同じ方法で自動化できる
    ・スプレッドシートが管理台帳になるので全体が見渡せる

  × やらない:
    ・自動いいね/自動フォロー
      → X・YouTube ともに規約違反でアカウント凍結リスク
      → 催眠チャンネルの「信頼感」を損なう

  △ 手動で行う:
    ・関連チャンネルへのコメント等の自然な交流
```

### 自動化が難しい工程

```
  以下はアプリ内の細かい操作が必要なため、完全自動化は難しい:
    ・GarageBand での EQ/コンプ/リバーブの微調整
    ・iMovie/DaVinci での編集操作
    → プリセット化・テンプレ化で省力化するのが現実的
```
