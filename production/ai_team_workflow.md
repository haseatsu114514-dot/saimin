## AIチーム編成

### チームメンバーと役割分担

```
┌─────────────────────────────────────────────────────────────────┐
│                    SHIZM AIプロダクションチーム                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  🔍 【リサーチャー】OpenCrawl                                    │
│  ├─ Web上のトレンド情報収集                                      │
│  ├─ 競合チャンネルの動向モニタリング                              │
│  ├─ YouTube検索トレンドの調査                                    │
│  └─ 視聴者コメントの傾向収集                                     │
│                                                                 │
│  📝 【プロデューサー / ライター】Claude                           │
│  ├─ 台本のドラフト生成                                          │
│  ├─ 企画の構成・設計                                             │
│  ├─ SEO最適化（タイトル・説明文・タグ）                           │
│  ├─ SNS投稿文の生成                                             │
│  ├─ 安全性チェック（医療表現の排除）                              │
│  └─ データ分析・改善提案                                         │
│                                                                 │
│  🎨 【アートディレクター】画像生成AI                              │
│  │  （Midjourney / DALL-E 3 / Stable Diffusion）                │
│  ├─ サムネイル用背景画像の生成                                   │
│  ├─ チャンネルアイコン・バナーの生成                              │
│  └─ 動画内の静止画背景の生成                                     │
│                                                                 │
│  🎵 【サウンドエンジニア補佐】音声AI                              │
│  │  （Adobe Podcast / Auphonic）                                │
│  ├─ AIノイズ除去・音声クリーンアップ                              │
│  └─ 自動ラウドネス調整                                          │
│                                                                 │
│  🎤 【パフォーマー / 最終決定者】自分                             │
│  ├─ 台本の最終調整・声に馴染ませる                               │
│  ├─ 音声収録（読み上げ）                                        │
│  ├─ 品質の最終チェック                                          │
│  └─ チャンネル全体の方針決定                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 各AIの選定理由と使い分け

| AI | 主な役割 | 選定理由 | コスト目安 |
|---|---|---|---|
| **Claude** | 台本生成・SEO・分析 | 長文の構造的な台本生成に強い。安全性配慮の指示に忠実。日本語の自然さ。 | Pro: $20/月 |
| **OpenCrawl** | Web調査・トレンド収集 | リアルタイムのWeb情報にアクセス可能。競合調査の自動化。 | 要確認 |
| **Midjourney** | サムネ・背景画像 | 高品質なアート生成。幻想的・スピリチュアルな画風に強い。 | $10/月〜 |
| **DALL-E 3** | 補助的な画像生成 | ChatGPT経由で手軽に使える。プロンプトの解釈が正確。 | ChatGPT Plus内 |
| **Adobe Podcast** | 音声ノイズ除去 | AIベースのノイズ除去が非常に高品質。無料枠あり。 | 無料〜 |
| **Canva** | サムネ仕上げ | テキストオーバーレイ・レイアウト調整。テンプレート管理。 | 無料〜Pro ¥1,000/月 |

## ツール間の連携フロー

```
 【企画フェーズ】
  OpenCrawl ──(トレンド情報)──▶ Claude ──(企画書・構成案)──▶ 自分が承認

 【台本フェーズ】
  Claude ──(台本ドラフト)──▶ 自分が修正 ──(修正指示)──▶ Claude ──(最終台本)

 【ビジュアルフェーズ】
  Claude ──(画像プロンプト)──▶ Midjourney/DALL-E ──(生成画像)──▶ Canva ──(サムネ完成)

 【音声フェーズ】
  自分が収録 ──(生音声)──▶ Adobe Podcast ──(ノイズ除去済み音声)──▶ GarageBand ──(最終ミックス)

 【公開フェーズ】
  Claude ──(タイトル・説明文・タグ・SNS投稿文)──▶ 自分がYouTubeにアップロード

 【分析フェーズ】
  OpenCrawl ──(視聴データ・コメント収集)──▶ Claude ──(分析・改善提案)──▶ 次回の企画へ
```

      ┃ 画像/サムネ生成    ┃ Midjourney + Canva  ┃ 所要: 30分
━━━━━━╋━━━━━━━━━━━━━━━━━━╋━━━━━━━━━━━━━━━━━━━╋━━━━━━━━━━━
