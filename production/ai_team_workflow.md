## AIチーム（4AI + 自分）

```
┌──────────────────────────────────────────────────────────────┐
│                 SHIZM AI プロダクションチーム                   │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  【ライター】Claude                                          │
│  ├─ 台本のドラフト生成（リポジトリのノウハウ参照）               │
│  ├─ 構成設計・4フェーズの時間配分                              │
│  └─ 安全性チェック・分析・改善提案                             │
│                                                              │
│  【企画・SEO・SNS】ChatGPT                                   │
│  ├─ テーマのブレスト・壁打ち・Web検索                          │
│  ├─ タイトル・説明文・タグ・チャプター生成                      │
│  └─ SNS投稿文の生成（→ Buffer/Zapierで自動投稿）              │
│                                                              │
│  【アートディレクター】ナナバナ（Gemini）                       │
│  └─ サムネ・背景画像・アイコン・バナーの生成                    │
│                                                              │
│  【サウンドエンジニア】Adobe Podcast                           │
│  └─ AIノイズ除去・ラウドネス調整                               │
│                                                              │
│  【パフォーマー / 最終決定者】自分                              │
│  ├─ 台本の最終調整・読み上げ収録                               │
│  └─ 各工程の最終チェック・GO判断                               │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### Claude vs ChatGPT の使い分け

```
  Claude                            ChatGPT
  ─────────────────────             ─────────────────────
  台本を書く           ◀──────▶     アイデアを広げる
  構成を設計する       ◀──────▶     SEO・メタデータを作る
  安全性を確認する     ◀──────▶     SNS投稿文を作る
  分析・改善する       ◀──────▶     壁打ち・Web検索

  【原則】
  ・「台本を書く」→ Claude（リポジトリ参照）
  ・「それ以外のテキスト」→ ChatGPT
  ・迷ったら両方に投げて良いほうを採用
```

## 動画1本の制作フロー（全12工程）

```
工程    やること                    担当              備考
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【企画】
 1.    テーマ・構成決定            ChatGPT + Claude
       ・トレンド調査/ブレスト      → ChatGPT（Web検索）
       ・タイトル・SEO候補生成      → ChatGPT
       ・構成プラン・時間配分       → Claude
       ・自分がGOを出す            → 【自分】

【台本】
 2.    台本ドラフト生成            Claude            リポジトリのテンプレ参照
                                                    安全性チェックも込み

 3.    台本の読み合わせ・修正       【自分】 + Claude
       ・声に出して読む            → 【自分】
       ・修正指示→反映            → Claude
       ・最終OK                   → 【自分】

【ビジュアル】（※台本確定後〜収録と並行可）
 4.    サムネ・背景画像の生成       ナナバナ(Gemini)   複数パターン生成
 5.    サムネ仕上げ               Canva + 【自分】   テキスト配置→選定

【収録・編集】
 6.    音声収録                   【自分】           GarageBandで録音

 7.    音声処理                   Adobe Podcast
                                 + GarageBand
       ・ノイズ除去/ラウドネス      → Adobe Podcast
       ・EQ/コンプ/リバーブ/環境音  → GarageBand

 8.    動画書き出し               iMovie/DaVinci     静止画+音声→動画

【公開】
 9.    公開メタデータ生成          ChatGPT           タイトル確定・説明文・
                                                    タグ・チャプターを一括生成

10.    アップロード・公開          【自分】           YouTube にアップ

【告知】（※自動化）
11.    SNS告知                    自動               ChatGPT で投稿文生成
                                                    → Buffer/Zapier で自動投稿

【振り返り】（※次の動画の工程1と兼ねてもOK）
12.    分析・次回改善              ChatGPT + Claude
       ・視聴データ/コメント確認    → ChatGPT（Web検索）
       ・改善提案・次テーマ検討     → Claude
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 集計

```
全12工程の内訳:
  AI がやる  ─── 6工程（工程 1, 2, 4, 9, 11, 12）
  自分がやる ─── 3工程（工程 3, 6, 10）
  両方      ─── 3工程（工程 5, 7, 8）

自分が必ずやること:
  ・台本を声に出して読む → 修正する（3）
  ・音声を収録する（6）
  ・YouTubeにアップロードする（10）

それ以外はすべてAIまたはツールが処理。
```

## ツール間の連携フロー

```
 【企画〜台本】
  ChatGPT ──(テーマ・タイトル案)──▶ Claude ──(構成・台本)──▶ 自分が読む・修正
                                         └──(修正反映)──▶ 台本確定

 【ビジュアル】
  ナナバナ ──(生成画像)──▶ Canva ──(テキスト配置)──▶ サムネ完成

 【音声〜動画】
  自分が収録 ──▶ Adobe Podcast ──▶ GarageBand ──▶ iMovie/DaVinci ──▶ 動画完成

 【公開〜告知】
  ChatGPT ──(メタデータ一式 + SNS投稿文)──▶ YouTube公開（自分）
                                            └──▶ Buffer/Zapier ──▶ X/コミュニティ自動投稿

 【振り返り】
  ChatGPT ──(データ確認)──▶ Claude ──(分析・改善)──▶ 次の企画へ
```

## SNS自動化の方針

```
  ◎ 自動化する:
    ・投稿文の生成 → ChatGPT
    ・予約投稿/自動投稿 → Buffer または Zapier
    ・YouTube公開 → X自動告知の連携 → Zapier/Make

  × やらない:
    ・自動いいね/自動フォロー
      → X・YouTubeともに規約違反でアカウント凍結リスク
      → 催眠チャンネルの「信頼感」を損なう

  △ 手動で行う:
    ・関連チャンネルへのコメント等の自然な交流
```
