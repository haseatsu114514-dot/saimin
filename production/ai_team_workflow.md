### 各AIの選定理由

| AI | 役割 | なぜこのAIか | コスト目安 |
|---|---|---|---|
| **Claude** | 台本生成・構成設計 | リポジトリ内のノウハウ（テンプレ・構成パターン・チェックリスト）を参照しながら長文台本を生成できる。指示への忠実さと安全性配慮。 | Pro: $20/月 |
| **ChatGPT** | ブレスト・SEO | アイデアの発散に強い。タイトル案の大量比較やSEO最適化に向く。 | Plus: $20/月 |
| **ナナバナ（Gemini）** | 画像生成 | サムネ・背景・アイコン等のビジュアル制作。 | 要確認 |
| **Codex** | 自動化・リポジトリ管理 | コード変更・ファイル操作をPR単位で非同期に依頼可能。テンプレ整備やフォーマット変換を任せられる。 | Pro内 |
| **OpenCrawl** | Web調査・トレンド | リアルタイムのWeb情報にアクセス。競合調査の自動化。 | 要確認 |
| **Adobe Podcast** | 音声ノイズ除去 | AIベースのノイズ除去が高品質。 | 無料〜 |
| **Canva** | サムネ仕上げ | ナナバナで生成した画像にテキストオーバーレイ。テンプレート管理。 | 無料〜 |

### Claude vs ChatGPT の使い分け方針

```
                Claude                          ChatGPT
  ─────────────────────────────    ─────────────────────────────
  台本のドラフト生成 ◀──得意──    ──得意──▶ タイトル案のブレスト
  構成設計・時間配分 ◀──得意──    ──得意──▶ SEO・タグ・説明文
  安全性チェック     ◀──得意──    ──得意──▶ SNS投稿文
  データ分析・改善   ◀──得意──    ──得意──▶ アイデアの壁打ち
  リポジトリ参照     ◀──得意──

  【原則】
  ・「台本を書く」仕事 → Claude（リポジトリのノウハウを活用）
  ・「アイデアを広げる・見栄えを作る」仕事 → ChatGPT
  ・迷ったら両方に投げて良いほうを採用（コンペ方式）
```

## ツール間の連携フロー

```
 【企画】
  OpenCrawl ──(トレンド情報)──▶ ChatGPT ──(アイデア・タイトル案)──▶ 自分が選択
                                    │
                                    └──(テーマ確定)──▶ Claude ──(構成プラン)

 【台本】
  Claude ──(台本ドラフト)──▶ 自分が読む・修正 ──(修正指示)──▶ Claude ──(最終台本)

 【ビジュアル】
  Claude ──(画像プロンプト案)──▶ ナナバナ(Gemini) ──(生成画像)──▶ Canva ──(サムネ完成)

 【音声】
  自分が収録 ──(生音声)──▶ Adobe Podcast ──(クリーン音声)──▶ GarageBand ──(最終ミックス)

 【公開】
  ChatGPT ──(タイトル・説明文・タグ・SNS投稿文)──▶ 自分がアップロード・投稿

 【分析】
  OpenCrawl ──(データ収集)──▶ Claude ──(分析・改善)──▶ 次の企画へ

 【リポジトリ管理】
  Codex ──(テンプレ更新・スクリプト整備)──▶ リポジトリに反映
```
