# PC画面操作 自動化手順書

Claude Computer Use（こわーく）で自動化する作業の手順書。
各作業をClaude に依頼する際のテンプレートとして使用する。

---

## 1. YouTube Studio への動画アップロード

### 事前準備
- YouTube Studio にブラウザでログイン済みであること
- 動画ファイルがローカルに保存済みであること
- スプレッドシート「SHIZM 制作管理」の「動画管理」シートにメタデータが記入済みであること
- サムネイル画像がローカルに保存済みであること

### Claude への指示テンプレート

```
以下の手順で YouTube Studio に動画をアップロードしてください。

1. YouTube Studio を開く（https://studio.youtube.com/）
2. 右上の「作成」→「動画をアップロード」をクリック
3. 動画ファイルを選択: [ファイルパス]
4. アップロード中に以下を入力:
   - タイトル: スプレッドシート「動画管理」の該当行からコピー
   - 説明文: 同上
   - サムネイル: [サムネイルのファイルパス] をアップロード
5. 「その他のオプション」→「タグ」にタグを入力
6. 視聴者 → 「いいえ、子ども向けではありません」を選択
7. 公開設定:
   - [すぐに公開 / 予約公開（日時: YYYY-MM-DD HH:MM）]
8. 「公開」をクリック
9. スプレッドシートの「状態」を「公開済み」に更新
```

## 2. X（Twitter）への告知投稿

### 事前準備
- X にブラウザでログイン済みであること
- スプレッドシート「SNS投稿」シートに投稿文が記入済みであること

### Claude への指示テンプレート

```
以下の手順で X に告知を投稿してください。

1. スプレッドシート「SNS投稿」シートを開く
2. 動画ID [XXX] の行から「X投稿文」をコピー
3. X（https://x.com/）を開く
4. 投稿画面に投稿文を貼り付け
5. YouTube動画のURLが含まれていることを確認
6. 「ポスト」をクリック
7. スプレッドシートの「投稿済み」を「はい」に更新
```

## 3. YouTube Studio からの分析データ取得

### Claude への指示テンプレート

```
以下の手順で YouTube Studio の分析データをスプレッドシートに転記してください。

1. YouTube Studio を開く
2. 左メニュー「アナリティクス」をクリック
3. 「コンテンツ」タブを開く
4. 各動画について以下のデータを確認:
   - 視聴回数
   - 高評価数
   - コメント数
   - インプレッションのクリック率（CTR）
   - 平均視聴維持率
   - インプレッション数
5. スプレッドシート「分析ログ」シートを開く
6. 該当する動画の行にデータを転記
```

## 4. スプレッドシートへのメタデータ記入

ChatGPTで生成したテキストをスプレッドシートに記入する手順。

### Claude への指示テンプレート

```
以下の内容をスプレッドシート「動画管理」シートに記入してください。

動画ID: [XXX]
タイトル: [タイトル]
説明文: [説明文]
タグ: [タグ]
チャプター: [チャプター]
公開日: [YYYY-MM-DD]
公開時間: [HH:MM]
状態: 準備中
```

## 5. コミュニティ投稿

### Claude への指示テンプレート

```
以下の手順で YouTube コミュニティに投稿してください。

1. YouTube Studio を開く
2. 左メニュー「コンテンツ」→ 上部タブ「コミュニティ」をクリック
3. 投稿画面にスプレッドシート「SNS投稿」の「コミュニティ投稿文」を貼り付け
4. 「投稿」をクリック
```

---

## 注意事項

- 各作業の前に、ブラウザで該当サービスにログイン済みであることを確認する
- 投稿前に内容を画面上で確認し、問題がなければ実行する
- エラーが発生した場合は、スクリーンショットを撮って報告する
- **自動いいね・自動フォローは絶対に行わない**（規約違反）
