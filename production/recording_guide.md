## 録音環境のセットアップ

### 部屋の選択

| 優先度 | 環境 | メリット |
|---|---|---|
| ◎ | クローゼット内 | 衣服が吸音材の役割。反響が最も少ない |
| ○ | 6畳以下の部屋 + リフレクションフィルター | 手軽。フィルターで最低限の反響対策 |
| △ | 広い部屋 | 反響が多い。布や毛布で壁を覆う等の対策が必要 |

### ノイズ対策チェックリスト

録音前に必ず確認：

- [ ] エアコンの電源を切った
- [ ] 冷蔵庫の音が入らない位置に移動した（または冷蔵庫の電源を一時的にOFF）
- [ ] 窓を閉めた
- [ ] スマホをサイレントモードにした（通知音防止）
- [ ] 時計のカチカチ音がない
- [ ] PCのファン音が入らない距離にマイクを設置した
- [ ] 近隣の工事・交通音が少ない時間帯に録音する

> ⚠️ **催眠音声はノイズに特に敏感**。リスナーはイヤホン・ヘッドホンで集中して聴くため、わずかなノイズも気になる。

## マイクセッティング

### 距離と角度

| 項目 | 推奨値 |
|---|---|
| マイクとの距離 | 15〜25cm |
| 角度 | 正面からやや斜め（破裂音対策） |
| ポップフィルターの位置 | マイクから5〜10cm手前 |

### ゲイン設定

- ピークが **-12dB〜-6dB** の範囲に収まるように調整
- ささやき声で録音するため、ゲインはやや高めになるが、クリッピングしないこと
- テスト録音で最大音量の箇所を確認してからゲインを決定

---

## Scarlett Solo の設定

1. USBでMacに接続
2. 48Vファンタム電源をONにする（AT2020はファンタム電源が必要）
3. INST/LINEスイッチを **LINE** に設定
4. AIRモードは **OFF**（催眠音声には不要。高域が強調されすぎる）
5. ダイレクトモニターは **ON**（自分の声をリアルタイムで確認）

## GarageBand 録音設定

### 新規プロジェクトの作成

1. GarageBandを開く → 「空のプロジェクト」
2. オーディオトラックを選択 → 入力をScarlett Soloに設定
3. テンポ・拍子は気にしなくてOK（自由テンポの語り）

### 録音時の推奨設定

| 項目 | 設定値 |
|---|---|
| サンプルレート | 44.1kHz（YouTube用には十分） |
| ビット深度 | 24bit |
| 入力モニタリング | ON |

## 録音のコツ

### 声の出し方

- **ささやき声〜普通の声の中間**。完全なウィスパーは聴き取りづらい
- 低めのトーンを意識（胸に響かせるイメージ）
- **間を恐れない**。催眠誘導は沈黙が重要。3〜7秒の間を積極的に入れる
- 台本の「……」は2〜3秒の間を意味する

### 収録の進め方

1. テスト録音（30秒）→ ゲインと音質を確認
2. 台本を最初から最後まで通しで読む（途中で止めない）
3. 間違えた箇所は3秒の沈黙を置いてから言い直す（編集で切りやすくなる）
4. 録音後すぐにヘッドホンで全体を聴き返す

### NGパターン

| NG | 理由 |
|---|---|
| 早口 | リスナーがリラックスできない |
| 声が高すぎる | 落ち着いた印象が出ない |
| 棒読み | 「読んでいる」感が出て誘導に入れない |
| 不自然な強調 | 催眠は自然な語りが命 |

## 録音後のチェックリスト

- [ ] クリッピング（音割れ）がないか
- [ ] ノイズ（エアコン、冷蔵庫等）が入っていないか
- [ ] 間の長さが適切か
- [ ] 声のトーンが全体を通して安定しているか
- [ ] 言い間違いの箇所をメモしたか（編集時に使用）
