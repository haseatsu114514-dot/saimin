## テンプレート A：「考えすぎを手放す催眠誘導」

> ※ Michael Sealey最大ヒットの日本語版ポジション。第1本目の推奨テーマ。

### 1. 導入（約7分）

```
【動画冒頭・ナレーション】
この催眠誘導は、考えすぎてしまう心を穏やかに鎮め、
思考のループから解放されるために作られたものです。
楽な姿勢で、リラックスして聴いてください。

---

それでは……楽な姿勢をとって……
椅子に座っていても、横になっていても構いません……
目を閉じてください……

（間3秒）

深呼吸をしましょう……
鼻から……深く……吸い込んで……
口から……長く……ゆっくりと……吐き出す……

（間3秒）

もう一度……吸い込んで……
吐き出す時に……頭の中の雑念も……一緒に外に出すイメージで……
ゆっくりと……吐き出してください……

（間3秒）

今、頭の中にいろんな考えが浮かんでいるかもしれません……
それはまったく自然なことです……
無理に止めようとしなくて大丈夫です……

ただ……これから少しの間……
その考えを……横に置いてみましょう……
消すのではなく……ただ、一旦……置いておくだけです……

（間5秒）
```

### 3. 暗示 — 思考を手放す（約15分）

```
今……あなたの目の前に……
静かな川が流れています……

穏やかで……澄んだ水が……
ゆっくりと流れていく川です……

（間5秒）

あなたの頭の中に浮かんでくる考え……
一つひとつを……木の葉に乗せて……
この川に流してみましょう……

（間3秒）

考えが浮かんだら……
「あ、また考えが浮かんだな」と……ただ気づいて……
それを木の葉に乗せて……川に流す……

（間5秒）

考えの内容は、なんでも構いません……
仕事のこと……人間関係のこと……将来の不安……
過去の後悔……何であっても……

ただ……葉っぱに乗せて……流していきます……

（間7秒）

流した考えは……川の向こうへ……
ゆっくりと……遠ざかっていきます……
追いかける必要はありません……

（間5秒）

[★ 以下の暗示を、表現を変えながら繰り返す]

あなたは考えを手放すことができます……
手放したあとに残るのは……静けさです……
その静けさの中に……安心があります……

考えることと、考えに支配されることは違います……
あなたはいつでも……考えを手放す力を持っています……

（間7秒）

川の流れを見ているうちに……
頭の中がだんだん静かになっていくのを感じてください……

考えの隙間に……穏やかな空白が生まれています……
その空白は……心地よいものです……

（間7秒）

[※ さらに暗示を展開し、合計15分程度に調整]
```

## テンプレート B：「自己肯定感を高める催眠」

### 暗示パートの方向性

```
[導入・深化は共通テンプレートを使用]

暗示パートのイメージ：
- 鏡の部屋に入り、自分自身と向き合う
- 過去の自分（子どもの頃の自分）と出会い、認めてあげる
- 「あなたはそのままで十分です」系の暗示
- 内なる批判的な声を、優しい声に変えていく

キーフレーズ例：
「あなたは……そのままで……十分に価値がある存在です……」
「完璧でなくていい……今の自分を……認めてあげましょう……」
「誰かと比べる必要はありません……
 あなたはあなたのペースで……成長しています……」
```

## テンプレート C：「ネガティブ思考を手放す催眠」

### 暗示パートの方向性

```
暗示パートのイメージ：
- 胸の中の黒い煙（ネガティブな感情）を、呼吸で少しずつ吐き出す
- 代わりに、金色の光を吸い込む
- 体の中が光で満たされていくイメージ

キーフレーズ例：
「ネガティブな考えは……あなたの一部ですが……
 あなたのすべてではありません……」
「手放しても大丈夫です……
 手放した分だけ……新しい光が入ってきます……」
```
