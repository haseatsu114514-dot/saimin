# 催眠音声の基本構成パターン

催眠音声は基本的に **4つのフェーズ** で構成される。すべてのテンプレートはこの構造をベースにカスタマイズして作成する。

---

## 4フェーズ構造

```
┌─────────────────────────────────────────────────────┐
│  1. 導入（Induction）        約5〜10分              │
│  ─────────────────────────────────────────────────  │
│  2. 深化（Deepening）        約5〜10分              │
│  ─────────────────────────────────────────────────  │
│  3. 暗示（Suggestion）       約10〜30分             │
│  ─────────────────────────────────────────────────  │
│  4. 覚醒（Awakening）        約3〜5分               │
│     ※ 睡眠導入の場合は覚醒なし（そのまま眠りへ）    │
└─────────────────────────────────────────────────────┘
```

---

## フェーズ1：導入（Induction）

### 目的
リスナーの日常意識からリラックス状態への移行を促す。

### 基本要素
- **安全な空間の提示**: 「楽な姿勢で…目を閉じて…」
- **呼吸の誘導**: 深呼吸のカウント（3〜5回）
- **身体のリラックス**: 全身の力を抜くスキャン（頭→肩→腕→脚…）
- **日常からの切り離し**: 「今日のことは一旦横に置いて…」

### 声のトーン
- 落ち着いた、ゆっくりとした語り
- 日常会話よりもやや低め、穏やかに
- 最初はリスナーの意識を引きつける適度な明瞭さ

### テンプレート

```
それでは、楽な姿勢をとって……
目を閉じても、薄く開けたままでも構いません……

まず、深く息を吸い込んで……
そして、ゆっくりと……吐き出していきましょう……

（間3秒）

もう一度……深く……吸い込んで……
そして……ゆっくりと……吐き出す……

（間3秒）

呼吸を続けるたびに……
体がだんだんとリラックスしていくのを感じてください……

頭のてっぺんから……柔らかい光が降り注ぐように……
首……肩……腕……と……
順番にリラックスが広がっていきます……
```

---

## フェーズ2：深化（Deepening）

### 目的
リラックス状態をさらに深め、潜在意識にアクセスしやすい状態を作る。

### 基本テクニック

| テクニック | 説明 | 向いているテーマ |
|---|---|---|
| **階段下降法** | 10から1のカウントダウンで階段を降りるイメージ | 汎用的 |
| **エレベーター法** | エレベーターで深い階へ降りていく | メンタルケア系 |
| **森の小道法** | 森の中の小道を歩いていく | 安眠系 |
| **波のリズム法** | 波の音・リズムに合わせて深まる | 安眠系 |
| **光のシャワー法** | 温かい光が体を包んでいく | スピリチュアル系 |

### テンプレート（階段下降法）

```
今から、10の階段をゆっくりと降りていきます……
一段降りるごとに……さらに深いリラックスの世界へ……

10……一歩降りて……少しだけ深くなります……
9……もう一歩……さらに心地よく……
8……体がさらに重く……柔らかく……
7……周りの音がだんだん遠くなって……
6……半分まで来ました……とても心地よい状態です……
5……さらに深く……安全な場所へ……
4……意識がぼんやりとして……
3……あと少し……もっと深く……
2……ほとんど深い場所に到着しました……
1……完全に……深い……リラックスの世界に……到着しました……

（間5秒）
```

---

## フェーズ3：暗示（Suggestion）

### 目的
テーマに応じた暗示・イメージ誘導を行うメインパート。

### テーマ別アプローチ

| テーマ | 暗示の方向性 |
|---|---|
| 安眠 | 眠りに落ちていく安心感、温かいベッドのイメージ |
| 考えすぎを手放す | 思考を雲に乗せて流す、川に流すイメージ |
| 自己肯定感 | 過去の自分を認める、内なる声との対話 |
| 前世退行 | 時間の回廊を遡る、扉を開けるイメージ |
| ネガティブ浄化 | 黒い煙を吐き出す、光で洗い流すイメージ |

### 暗示の基本原則

1. **肯定形を使う**: 「不安がなくなる」ではなく「安心が広がる」
2. **五感を使う**: 視覚・聴覚・触覚のイメージを織り交ぜる
3. **選択の余地を残す**: 「〜かもしれません」「〜と感じる人もいます」
4. **繰り返し**: 重要な暗示は表現を変えて2〜3回繰り返す
5. **間を多く**: 暗示のあとに沈黙の時間を入れ、潜在意識に浸透させる

---

## フェーズ4：覚醒（Awakening）

### 目的
リスナーを通常の意識状態に安全に戻す。

> **注意**: 睡眠導入系の場合、覚醒パートは入れない（そのまま眠りに移行）。代わりに「このまま……心地よい眠りの中へ……おやすみなさい……」で終える。

### テンプレート

```
それでは……少しずつ……
日常の意識に戻っていきましょう……

1……足先に意識が戻ってきます……
2……手の指先が動き始めます……
3……体全体に力が戻ってきます……
4……周りの音が聞こえてきます……
5……目を開けて……すっきりと目覚めます……

（間3秒）

お疲れさまでした……
今日体験したことを……心の奥で大切にしてください……
```

## 時間配分の目安

| 動画タイプ | 導入 | 深化 | 暗示 | 覚醒 | 合計 |
|---|---|---|---|---|---|
| ショート（20分） | 5分 | 3分 | 10分 | 2分 | 20分 |
| スタンダード（30分） | 7分 | 5分 | 15分 | 3分 | 30分 |
| ロング（45分） | 10分 | 7分 | 25分 | 3分 | 45分 |
| 超ロング（60分） | 10分 | 10分 | 35分 | 5分 | 60分 |
| 睡眠導入（30分） | 7分 | 5分 | 18分 | なし | 30分 |

---

## 声の演出ガイド

| フェーズ | 声のトーン | テンポ | 間の長さ |
|---|---|---|---|
| 導入 | 穏やか・明瞭 | ゆっくり | 2〜3秒 |
| 深化 | さらに穏やか・低め | とてもゆっくり | 3〜5秒 |
| 暗示 | 内容による（力強い↔柔らかい） | 内容による | 3〜7秒 |
| 覚醒 | 徐々に明るく・明瞭に | 少しずつ速く | 2〜3秒 |

## 台本作成チェックリスト

- [ ] 4フェーズの構成になっているか
- [ ] 導入で安全な空間を提示しているか
- [ ] 深化テクニックが適切に選ばれているか
- [ ] 暗示が肯定形で書かれているか
- [ ] 五感のイメージが含まれているか
- [ ] 間の指示（秒数）が入っているか
- [ ] 覚醒パートが安全か（睡眠導入の場合は覚醒なしか）
- [ ] 声に出して読み、不自然な箇所がないか
- [ ] 全体の時間が目標に合っているか
- [ ] 免責事項（医療行為ではない旨）を動画概要欄に入れる準備があるか
