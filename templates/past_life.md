# 前世退行催眠 台本テンプレート

> 構成の詳細は [structure_patterns.md](./structure_patterns.md) を参照

## テンプレート概要

| 項目 | 内容 |
|---|---|
| ジャンル | 柱3：意識探索系催眠 |
| 想定時間 | 45〜60分（探索パートが長い） |
| 覚醒パート | **あり**（安全に戻す） |
| 深化テクニック | 光のシャワー法 → 時間の回廊 |
| ターゲット | スピリチュアルに興味がある人、自己理解を深めたい人 |

---

## 重要な注意事項

> ⚠️ 前世退行催眠は、リスナーが強い感情体験をする可能性がある。以下の点を必ず動画内・概要欄で伝える：
> - これはエンターテインメント・自己探索体験であり、医療行為ではない
> - 精神的に不安定な時期の方は、専門家に相談してから体験すること
> - 不快に感じたらいつでも目を開けて終了してよい

---

## 台本テンプレート

### 1. 導入（約10分）

```
【動画冒頭・ナレーション】
この催眠誘導は、あなたの深い意識の中にある記憶……
前世の記憶と呼ばれるものにアクセスする体験です。

これは一つの自己探索の旅です。
何かが見えても、見えなくても、どちらでも構いません。
大切なのは、リラックスして、この体験を楽しむことです。

もし途中で不快に感じたら、いつでも目を開けて
この体験を終わりにしてください。あなたの安全が第一です。

---

[共通の導入テンプレートを使用]
[通常よりも丁寧に、時間をかけてリラックスさせる]
```

### 2. 深化 — 光のシャワー＋時間の回廊（約10分）

```
今、あなたの頭上から……
温かい白い光が……降り注いでいます……

その光は……あなたの体全体を包んで……
あなたを守る繭のようになっています……

（間5秒）

この光に包まれている限り……
あなたは完全に安全です……
どこにいても……いつの時代にいても……
安全なまま……観察することができます……

（間5秒）

今、目の前に……長い廊下が現れます……
時間の回廊です……

壁には、いくつもの扉があります……
それぞれの扉の向こうに……
あなたの魂が経験した時代の記憶が眠っています……

（間5秒）

ゆっくりと……この廊下を歩いていきましょう……
一つの扉が……ほんのりと光っているのが見えます……
その扉が……今のあなたに必要な記憶へとつながっています……

（間5秒）

その扉に手をかけて……
ゆっくりと……開けてみましょう……

（間7秒）
```

### 3. 暗示 — 前世の体験（約25分）

```
扉の向こうに……光が溢れています……
一歩、踏み出してみましょう……

（間5秒）

まず……自分の足元を見てください……
何か見えますか？……
靴を履いていますか？……裸足ですか？……
どんな地面の上に立っていますか？……

（間10秒）

次に……体を見てみましょう……
どんな服を着ていますか？……
男性ですか？……女性ですか？……
年齢はどれくらいに感じますか？……

（間10秒）

周りを見渡してみてください……
どんな場所にいますか？……
建物が見えますか？……自然の中ですか？……
天気はどうでしょう？……

（間10秒）

[★ 以下、探索を進める質問を続ける]

近くに……誰かいますか？……
その人はどんな人ですか？……
あなたとどんな関係の人ですか？……

（間10秒）

今のあなたは……何をしていますか？……
どんな気持ちですか？……

（間10秒）

[※ ここでは沈黙の時間を長めに取る]
[※ リスナーが自分のイメージを構築する時間が必要]

この体験から……
今のあなたの人生に……
何かメッセージがあるとしたら……
それは何でしょうか？……

心の中で……そのメッセージを受け取ってください……

（間15秒）

[※ 十分な間をとったあと、戻りの準備に入る]

そろそろ……この場所を離れる時間です……
見たもの、感じたことを……心に留めて……
感謝の気持ちとともに……時間の回廊に戻りましょう……

扉をくぐって……回廊に戻ります……

（間5秒）

光の繭があなたを守りながら……
現在の時代へ……戻ってきます……
```

### 4. 覚醒（約5分）

```
時間の回廊が……少しずつ消えていき……
あなたは今……この部屋に戻ってきています……

体の感覚が戻ってきます……
布団の感触……周りの空気……

（間3秒）

[共通の覚醒テンプレートを使用、ただしよりゆっくりと]

※ 追加メッセージ：
「今日体験したことは……
あなたの潜在意識からのメッセージです……
正しいか間違いかではなく……
感じたことを……大切にしてください……」
```

## 制作上の注意

| 項目 | ポイント |
|---|---|
| 間の長さ | 通常のテンプレートより長め（10〜15秒の沈黙も入れる） |
| 声のトーン | 神秘的だが押し付けがましくない |
| 誘導の仕方 | 「〜が見えます」ではなく「〜が見えるかもしれません」 |
| 安全への配慮 | 光の繭の暗示を随所に入れる |
| 免責事項 | 冒頭と概要欄に必ず記載 |
