# スプレッドシート管理台帳テンプレート

このフォルダのCSVファイルをGoogle スプレッドシートにインポートして使用する。

## インポート手順

1. Google スプレッドシートを新規作成（名前: 「SHIZM 制作管理」）
2. 各CSVを別シートとしてインポート:
   - `01_video_management.csv` → シート名「動画管理」
   - `02_sns_posts.csv` → シート名「SNS投稿」
   - `03_analytics_log.csv` → シート名「分析ログ」
3. インポート後、1行目をフリーズ（固定）する
4. 「状態」列にプルダウンを設定: `準備中` / `台本確定` / `収録済み` / `編集済み` / `公開済み`
5. 「投稿済み」列にプルダウンを設定: `はい` / `いいえ`

## 運用ルール

- 新しい動画を企画するたびに、動画管理シートに1行追加する
- ChatGPTで生成したメタデータ（タイトル・説明文・タグ等）はここに記入する
- Claude のPC画面操作で、このシートから読み取って YouTube Studio / X に入力する
- 公開後は分析ログシートにデータを転記する（Claude PC操作で自動化）
