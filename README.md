# SHIZM催眠部屋 | ヒプノセラピー — ナレッジベース

**SHIZM催眠部屋 | ヒプノセラピー** は、日本語圏で「催眠誘導（ヒプノセラピー）」に特化した男性声YouTubeチャンネルのプロジェクトです。

## コンセプト

**「男性の落ち着いた声で、意識の深い層に働きかける催眠誘導チャンネル」**

Michael Sealey（登録者210万人）が英語圏で確立したポジションの日本語版を目指します。

## コンテンツの3本柱

| 柱 | 内容 | 役割 |
|---|---|---|
| 🌙 安眠・睡眠導入催眠 | 寝落ち用催眠誘導（20分〜3時間） | 再生数の安定基盤 |
| 💆 メンタルケア催眠 | 考えすぎを手放す・自己肯定感・不安解消 | 幅広い層へのリーチ |
| 🔮 意識探索系催眠 | 前世退行・ハイヤーセルフ・五行瞑想 | 独自性・ファン化 |
